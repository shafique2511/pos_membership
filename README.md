# Luxantara Members

Private-use Membership + Booking + POS system for owner-managed businesses.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- ShadCN-style UI primitives
- Supabase Auth, PostgreSQL, Storage, and RLS

## Setup

1. Install dependencies:

   ```bash
   npm.cmd install
   ```

2. Copy `.env.example` to `.env.local` and fill in Supabase values.

3. Start the app:

   ```bash
   npm.cmd run dev
   ```

## Phase 1 Scope

This foundation includes routing, layouts, module-aware navigation, reusable UI components, base pages, and Supabase client setup. Database migrations and RLS are prepared for later phases.
