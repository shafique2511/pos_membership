# Owner Guide

The Owner has full control of the private-use business system.

## Owner Can

- Create and manage the business profile
- Select business type
- Apply preset suggestions when available
- Manually enable or disable modules
- Manage branches
- Manage staff permissions
- Manage customers, bookings, memberships, loyalty, POS, inventory, payments, reports, notifications, and marketing
- Access full Data Backup
- Export business data anytime

## Data Backup

Open:

`Settings > Data Backup`

Owner can:

- Backup all data
- Export by module
- Export by branch
- Export by date range
- Export reports as PDF, Excel, or CSV
- Download completed backups
- View backup history and status

Backup actions are audit logged.

## Important

The Owner owns the business data. The platform code and hosting process the data, but ownership remains with the business owner.
