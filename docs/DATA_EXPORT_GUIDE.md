# Data Export Guide

Owner can export business data anytime for operations, migration, shutdown preparation, or hosting move.

## Export Scopes

- Full business backup
- Customers
- Bookings
- Memberships
- Loyalty
- POS sales
- Inventory
- Staff
- Commission
- Payments
- Reports
- Marketing
- Settings
- Branch data

## Before Migration or Hosting Move

1. Open `Settings > Data Backup`.
2. Run `Backup All Data`.
3. Export key modules as CSV or JSON.
4. Export reports as PDF, Excel, or CSV if needed.
5. Download completed backups.
6. Store downloaded files securely.
7. Verify `backup_exports` and `audit_logs` contain records.

## Disabled Modules

Owner can still export old data from disabled modules. This protects data ownership and migration rights.
