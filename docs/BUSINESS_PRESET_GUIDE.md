# Business Preset Guide

Presets are starting suggestions only.

Preset-supported types:

- Barber Shop
- Coffee Shop
- Clinic
- Custom Business

No presets are created for:

- Salon
- Spa
- Event Space

## Changing Business Type

When changing business type, Owner chooses:

1. Keep current module settings
2. Apply preset defaults if selected type has a preset
3. Manually customize modules

If the selected type has no preset, the system uses manual module customization.

Owner can enable or disable modules after setup.
